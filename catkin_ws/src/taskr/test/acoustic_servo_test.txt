Welcome to the manual testing interface for acoustic_servo action!
--------
When prompted with `imu>` enter a value between 0 and 359.
This value represents the current heading of the robot.
When prompted with `png>` enter a value between 0 and 7.
This value represents the heading of the pinger.
0 coresponds to 0, 1 to 45, and so on for multiples of 45.
